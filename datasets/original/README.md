# Datasets — Original

Place your CRS-provided datasets here:

| File | Description | Used In |
|------|-------------|---------|
| `logo_dataset/` | Brand logo images with style labels | Module 1 |
| `font_dataset/` | Font image samples with metadata | Module 1 |
| `slogan_dataset.csv` | Brand slogans with industry/tone tags | Module 2 |
| `startups_dataset.csv` | Startup profiles with brand attributes | Modules 2, 4 |
| `marketing_campaign_dataset.csv` | Historical campaigns with CTR/ROI/Engagement | Module 3 |
